# `stonecut()`

此[方块函数](...md)返回方块切石后的结果。一个方块有多个切石结果时，会随机选择一个。例如，圆石可能被切石成圆石楼梯、圆石台阶或圆石墙，因此如果原方块为圆石，则方块应用的结果会是这三个方块中的随机一个。如果方块不能切石，则不进行操作。

切石后的方块会尽可能保留原有的方块状态属性。切石后拥有而原方块没有的属性，值会随机设置，除了 `waterlogged` 和 `powered`。例如，如果该函数应用于石头方块并返回得到石头楼梯，则楼梯的朝向和形状是随机的，但一定不含水。

## 语法

`stonecut([方块函数])`

## 参数

### `[方块函数]`

[方块函数](...md)，如果没有指定，则默认使用现有方块。
