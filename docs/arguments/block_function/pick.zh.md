# `pick()`：随机选择方块函数

此[方块函数](index.md)用于从多个方块函数中随机选择。可以为每个随机项设置权重，如果不设置，那么其权重均为 1。

## 语法

- `pick(<方块函数> [权重], ...)`
- `pick(<方块函数> [权重], ...; seed = <种子>)`

## 参数

### `<方块函数>`

至少一个需要应用的[方块函数](index.md)。

### `[权重]`

浮点数，0 到 1 之间。每个方块函数可能被选中的相对概率。权重越大的方块函数越有可能被选中。如果省略，则默认为 1。可以为 0，但所有方块函数的权重之和不能为 0。

### `<种子>`

长整型。用于随机确定各坐标采用的方块函数的种子。

## 示例

- `pick(white_wool, blue_wool)`：随机生成白色羊毛或蓝色羊毛，其概率相等。相当于 `white_wool|blue_wool`。
- `pick(white_wool, blue_wool; seed = 3)`：随机生成白色羊毛或蓝色羊毛，其概率相等。随机数种子为 3，因此如果对同一地方多次应用此方块函数，其生成的方块也会相同。
- `pick(white_wool 3, blue_wool 2)`：随机生成白色羊毛和蓝色羊毛，其白色羊毛的概率为 60%，蓝色羊毛的概率为 40%。
- `pick(white_wool 3, blue_wool)`：随机生成白色羊毛和蓝色羊毛，其白色羊毛的概率为 75%，蓝色羊毛的概率为 25%。
- `pick(white_wool, blue_wool 0)`：只会生成白色羊毛，因为蓝色羊毛的概率为 0。
- `pick(white_wool 0, blue_wool 0)`：无效的方块函数，因为各项的权重之和为 0。
