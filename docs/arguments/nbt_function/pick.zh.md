# `pick()`：随机选择 NBT 函数

此 [NBT 函数](...md)用于从多个 NBT 函数中随机选择。可以为每个随机项设置权重，如果不设置，那么其权重均为 1。

## 语法

- `pick(<NBT 函数> [权重], ...)`

## 参数

### `<NBT 函数>`

至少一个需要应用的 [NBT 函数](...md)。

### `[权重]`

浮点数，0 到 1 之间。每个 NBT 函数可能被选中的相对概率。权重越大的 NBT 函数越有可能被选中。如果省略，则默认为 1。可以为 0，但所有 NBT 函数的权重之和不能为 0。

## 示例

- `pick(123, 456)`：随机返回整数 `123` 或 `456` 中的一个，其概率相等。相当于 `123|456`。
- `pick(123 3, 456 2)`：随机返回整数 `123` 或 `456` 中的一个，其返回整数 `123` 的概率为 60%，`456` 的概率为 40%。
- `pick(123 0, 456 0)`：无效的 NBT 函数，因为各项的权重之和为 0。

## 数据结构

- `type`：此时为 `enhanced_commands:pick`。
- `functions`：复合标签。随机选择的 NBT 函数的列表。
    - `weighted`：布尔值。
    - `elements`：列表，仅当 `weighted` 为 false 时存在。
        - NBT 函数，一个被随机抽取的项。
    - `entries`：列表，仅当 `weighted` 为 true 时存在。
        - 复合标签。一个项。
            - `element`：NBT 函数。
            - `weighted`：双精度浮点数。
