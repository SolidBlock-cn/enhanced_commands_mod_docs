# `collector`

This [entity predicate](index.md) exists technically. When typing entity selectors, you can specify some [more entity selector types](../entity_selector.md#more-entity-selector-types), which collect entities in their own ways, and can be used as entity predicates.

This entity predicate does not support specifying through entity selector options, but is specified through entity predicate types.

## Data structure

- `type`: Currently `"collector"`.
- `collector`: String.
