# `probability()`：自己指定一个概率

只有在一定的概率下通过的[方块谓词](...md)。可以指定另一个方块谓词，只有在在一定的概率下且该方块谓词通过时通过。

## 语法

- `probability(<概率>, [方块谓词])`
- `probability(<概率>, [方块谓词], seed = <种子>)`

## 参数

### `<概率>`

浮点数，0 到 1 之间。

### `[方块谓词]`

指定需要同时满足的[方块谓词](...md)。如果指定了方块谓词，那么等效于 `probability(<概率>)*<方块谓词>`。如果未指定，则等价于 `probability(<概率>, *)`。

### `<种子>`

长整型。

## 示例

- `probability(0.35)`：只有 0.35 的概率通过。
- `probability(0.3, seed = 3)`：只有 0.3 的概率通过，其种子为 3。
- `probability(0.8, stone)`：只有 0.8 的概率，且方块为石头时通过。
- `probability(0.7, stone, seed = 5)`：只有 0.7 的概率，且方块为石头时通过，其种子为 5。

此方块谓词在实际应用时，可以制作随机的效果。例如，命令 `/replaceblocks sphere(20) probability(0.4, dirt) bedrock` 可以将 20 个方块内大约 40% 的泥土替换为基岩。
