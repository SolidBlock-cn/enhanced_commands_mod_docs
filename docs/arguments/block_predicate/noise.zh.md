# `noise()`

此[方块谓词](...md)用于根据噪声为每个位置决定方块谓词。

## 语法

- `noise(<方块谓词 1> [权重 1], <方块谓词 2> [权重 2], ...)`
- `noise(<方块谓词 1> [权重 1], <方块谓词 2> [权重 2], ...; seed = <种子>, first_octave = <主倍频>, amplitudes = <振幅>, scale = <x> [y] [z]; offset = <x> [y] [z])`

该方块函数的语法和参数均与 [`noise()` 方块函数](../../block_function/noise.md)一致，故这里不再详细描述。
