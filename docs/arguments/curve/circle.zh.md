# `circle()`：绘制一个任意的圆

沿指定的中心并按照指定的旋转轴向量的圆周或者圆周的一部分（圆弧），这个圆所在平面与旋转轴向量垂直。这种曲线有以下几个基本的属性：

- 半径：圆的半径，双精度浮点数。
- 中心：圆心的坐标。
- 轴：圆的旋转轴，是一个向量。圆所在平面将与这个向量垂直。
- 角度范围：圆的旋转范围。默认为 0 到 2π，即旋转完整的一周，从而绘制出完整的圆。

使用此曲线绘制的圆可以是任何大小、任何角度。与之相比，区域 [`hcyl()`](../region/cylinder.md) 只能绘制出以 y 轴为轴的完整的圆。

在绘制圆的过程中，会先根据半径和轴确定一个基础半径向量，基础半径向量的计算方法为：

- 计算轴与 y 轴正方向单位向量的向量积。
- 如果向量积非零向量，则将这个向量积单位化后乘以半径。
- 如果向量积为零向量（例如轴为零向量，或者轴与 y 轴平行），则：
    - 如果轴为零向量或 y 轴正方向向量，则取 x 轴正方向、长度等于半径的向量。
    - 如果轴为 y 轴负方向向量，则取 x 轴负方向、长度等于半径的向量。

确定了基础半径向量后，将其以圆心为旋转中心、以指定的轴为旋转轴，旋转一定的度数，这个度数是角度范围中的任意度数，从而形成一个圆或圆弧。

## 语法

- `circle(<半径>, [中心], pivot = [轴], range = [范围])`
- `circle(range = <半径>, center = [中心], pivot = [轴], range = [范围])`

其中，`range` 和 `center` 两个参数的参数名称可以省略，但是如果省略，则必须按顺序写。在明确指定了参数名称的情况下，各参数的顺序可以调换，但是不能重复。

## 参数

### `<半径>`

圆的半径，双精度浮点数。可以是任意数。如果是零，则会绘制出一个点。如果是负数，能够正常画出圆或圆弧。

### `[中心]`

坐标，可以是相对坐标或局部坐标。未指定时，则为命令源的坐标（例如玩家所在位置的坐标）。

### `[轴]`

圆的旋转轴。可以用以下方式指定：

- `<向量>`：直接用 3 个双精度浮点数指定轴向量，例如 `1 2 3`。
- `[长度] <方向>`：表示轴向量为指定的方向。`[长度]` 的默认值为 1，通常无需指定，因为作为轴时其长度不起作用（除非长度为 0）。
- `rotated <y> <x>`：指定偏航角和俯仰角，角度制（不带单位）。
- `facing <坐标>`：从命令源的位置指向指定的坐标的向量。

旋转轴没有指定时，默认为 y 轴正方向的单位向量。

### `[范围]`

旋转的角度范围，其格式为：

- `<起始角度>..<结束角度>`，两个角度的大小不做要求，会进行正确的比较。
- `<结束角度>`：相当于 `0..<结束角度>`。

其中，角度的格式为（均为双精度浮点数）：

- `0`：指定为零角度时可以不指定单位。
- `<数轴>deg`：角度制的角度，一个圆周为 360 度。
- `<数轴>rad`：弧度制的角度，一个圆周为 2π。
- `<数轴>turn`：指定圆周数，一个圆周为 1turn。

## 示例

- `circle(5)`：以当前坐标为中心、半径为 5、以 y 轴正方向为轴（即在水平面上）的完整圆。
- `circle(5, range = 0.5turn)`：北半圆弧。
- `circle(5, range = 0..0.5turn)`：与前一个例子等价。
- `circle(5, range = 0..180deg)`：与前一个例子等价。
- `circle(5, range = 0.25turn..0.75turn)`：西半圆弧。
- `circle(5, pivot = east)`：以 x 轴正方向为轴的完整圆周，即圆在 y、z 平面上。
- `circle(5, pivot = 0 1 2)`：绕 (0, 1, 2) 向量旋转的圆，该圆所在的平面与该向量垂直。
- `circle(5, pivot = rotated 45 20)`：朝向 (45, 20) 方向的圆。
- `circle(5, pivot = facing ^ ^ ^1)`：朝向玩家视角前方方向的圆，该圆所在的平面与玩家视角的方向垂直。
- `circle(5, center = ^^^5, pivot = facing ^^^5)`：玩家视角前方方向 5 格位置且以玩家视角方向为轴的圆，该圆所在的平面与玩家视角的方向垂直。
- `circle(5, ~ ~5 ~)`：以玩家上方 5 格的位置为中心、以 y 轴正方向为轴（即在水平面上）的圆。
- `circle(radius = 5, center = ~ ~5 ~)`：与前一个例子等价。

## 数据结构

- `type`：此时为 `"enhanced_commands:circle"`。
- `radius`：双精度浮点数。
- `center`：坐标。
- `pivot`：坐标。
- `min_angle`：双精度浮点数，可选，默认为 0。
- `max_angle`：双精度浮点数，可选，默认为 2π。
