# `single()`: Single block position

A [region](...md) that contains only a single block position.

## Syntax

- `<coordinate>`
- `single(<coordinate>)`

## Parameters

### `<coordinate>`

The single [block position](../../pos.md)。

## Examples

- `single(~~~)`
- `single(1 2 3)`
- `single(^^^5)`

## Data format

- **type**: String. Currently  `enhanced_commands:single`.
- **pos**: Array.
