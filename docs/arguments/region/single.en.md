# `single()`: Single block position

A [region](index.md) that contains only a single block position.

## Syntax

- `<coordinate>`
- `single(<coordinate>)`

## Parameters

### `<coordinate>`

The single [block position](../pos.md)。

## Examples

- `single(~~~)`
- `single(1 2 3)`
- `single(^^^5)`

## Data format

- **type**: Currently  `single`.
- **pos**: Array.
