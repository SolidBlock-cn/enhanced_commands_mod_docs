# `all()`：全都要满足，一个也不能少

多个[物品谓词](index.md)的交集。当多个物品谓词均通过时，这个物品谓词才通过。如果没有提供物品谓词，则始终通过。

## 语法

`all([物品谓词], ...)`

如果物品谓词多于 0 个，则等同于 `<物品谓词 1>&<物品谓词 2>` 的语法，参见[复合物品谓词](index.md#复合物品谓词)。

## 示例

- `all(#stairs, stone_stairs)`：当物品是楼梯且不是石头楼梯时通过。
- `all(stone, dirt)`：有效的物品谓词，但匹配不到任何物品，因为物品不可能既是石头又是泥土。
- `all()`：始终通过。等效于 `*`。

## 数据结构

- `type`：此时为 `"all"`。
- `predicates`：列表。
    - 一个物品谓词。
