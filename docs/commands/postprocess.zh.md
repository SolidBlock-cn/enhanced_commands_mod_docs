# `/postprocess` 和 `//postprocess`

此[命令](index.md)用于对方块进行后处理，例如修复栅栏与墙的连接问题。

## 语法

- `/postprocess <区域> [关键字参数]`
- `//postprocess [关键字参数]`

此命令在效果上等价于 `/etblocks <区域> postprocess()`，参见 [`postprocess()` 方块函数](../arguments/block_function/postprocess.md)。

此命令支持 [`/setblocks`](setblocks.md)的所有关键字参数。
