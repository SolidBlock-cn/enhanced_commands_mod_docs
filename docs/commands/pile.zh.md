# `/pile`

此[命令](...md)用于将多个实体通过骑乘的方式堆叠，即让多个实体依次骑乘。例如，如果选择器选择到 A、B、C、D 四个实体，那么就会让 B 骑乘 A、C 骑乘 B、D 骑乘 C。选择到的实体少于两个时，命令会执行失败。实体的排序方式取决于实体选择器的排序方式。

实体的骑乘会受到一些条件的限制。当部分实体骑乘失败时，不会影响其他实体的骑乘，有可能导致一次性形成多组堆叠的实体。

## 语法

- `/pile <目标>`

## 参数

### `<目标>`

[实体选择器](../../arguments/entity_selector.md)。命令执行时，会剔除到不在命令源所在的维度内的实体。选择到的实体少于 2 个时，命令会执行失败。

## 示例

- `/pile @n[type=chicken, limit=5]`：堆叠距离最近的 5 只鸡。
- `/pile @e[type=creeper|cat, sort=random, r=20]`：选择距离 20 格以内的所有苦力怕和猫，将其按照随机顺序堆叠。
- `/pile @r`：命令不会正常执行，因为选择到的实体少于 2 个。
